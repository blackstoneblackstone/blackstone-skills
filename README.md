# Blackstone Skills

个人 Qoder / OpenClaw 技能仓库。

## 下载技能

```bash
git clone https://github.com/blackstoneblackstone/blackstone-skills.git
```

技能目录结构：
```
blackstone-skills/
├── iu-comment-scraper/    # IU 评论抓取
└── sql-toolkit-1.0.0/     # SQL 工具包
```

## 提交技能

1. 创建技能目录：`{skill-name}-{version}/`
2. 编写 `SKILL.md` 和 `_meta.json`
3. 提交 PR 或推送到本仓库

## 技能列表

| 技能 | 说明 |
|------|------|
| iu-comment-scraper | 爬取 berriz.in 网站 IU 相关内容评论 |
| sql-toolkit-1.0.0 | SQL 数据库工具包（SQLite/PostgreSQL/MySQL） |
